Users need to be aware that once Covidien's Puritan Bennett 840 ventilators have received upgraded software for the neonatal configuration, a different procedure will usually be required when calibrating the units' oxygen sensor for neonatal use. Since the function of the previous "100% O2/cal 2 min" key--which some users employed for this calibration--has now changed, users should instead perform calibration through the "More Settings" menu.
